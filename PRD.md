# 회의실 예약 시스템 (Meeting Room Reservation System)

## 1. 프로젝트 개요

### 프로젝트명
Immersive_AI_service_project09 - 회의실 관리 시스템

### 목표
직관적인 UI를 통해 회의실의 예약 상황을 실시간으로 확인하고, 빈 시간대를 선택하여 즉시 예약할 수 있는 웹 기반 회의실 예약 시스템 구축

### 핵심 가치
- **즉시성**: 시간대 클릭으로 바로 예약 가능
- **가시성**: 모든 회의실의 예약 현황을 한 눈에 확인
- **간편성**: 복잡한 절차 없이 빠르고 쉬운 예약

---

## 2. 핵심 기능

### 2.1 예약 현황 보드
- **레이아웃**: 시간대(행) × 회의실(열) 그리드 형태
  - 수평축: 시간대 (예: 09:00 ~ 18:00, 30분 또는 1시간 단위)
  - 수직축: 회의실 1~6번
  
- **상태 표시**
  - 빈 시간대: 밝은 색상, 클릭 가능
  - 예약된 시간대: 다른 색상, 예약자/목적 표시
  - 현재 시간: 강조 표시

- **정보 표시**
  - 예약자명
  - 회의 목적/제목
  - 예약 시간 (시작~종료)

### 2.2 예약 시스템
- **예약 방법**: 빈 시간대 클릭
- **예약 프로세스**
  1. 빈 시간대(셀) 클릭
  2. 예약 정보 입력 모달/폼 표시
  3. 필수 정보 입력:
     - 예약자명
     - 회의 목적/제목
     - 회의실 번호 (자동 선택됨)
     - 날짜/시간 (자동 선택됨)
     - 예약 시간 길이 선택 (30분, 1시간 등)
  4. 확인 후 예약 완료

- **예약 취소**
  - 이미 예약된 시간대 클릭 시 상세 보기
  - 예약자가 취소 권한 있을 경우 취소 가능

### 2.3 회의실 관리
- **회의실 목록**: 총 6개 (1번~6번)
- **회의실 정보**
  - 회의실 번호
  - 위치 (추후 확장)
  - 수용 인원 (추후 확장)

## 3. 데이터 모델

### 3.1 Supabase 테이블 구조

#### `meeting_rooms` 테이블
```
- id (UUID, Primary Key)
- room_number (Integer, 1-6)
- room_name (String)
- capacity (Integer)
- created_at (Timestamp)
- updated_at (Timestamp)
```

#### `reservations` 테이블
```
- id (UUID, Primary Key)
- room_id (UUID, Foreign Key → meeting_rooms.id)
- reserver_name (String, 예약자명)
- meeting_title (String, 회의 목적/제목)
- start_time (Timestamp, 예약 시작 시간)
- end_time (Timestamp, 예약 종료 시간)
- status (Enum: 'confirmed', 'cancelled')
- created_at (Timestamp)
- updated_at (Timestamp)
```

### 3.2 데이터 관계
- 1개 회의실 : N개 예약 (1:N)

## 4. 사용자 시나리오

### 시나리오 1: 회의실 예약
1. 사용자가 예약 현황 보드에 접속
2. 원하는 회의실의 빈 시간대 확인
3. 빈 셀 클릭
4. 예약 정보 입력 (예약자명, 회의 목적 등)
5. 예약 버튼 클릭
6. 예약 완료 메시지 표시

### 시나리오 2: 예약 현황 확인
1. 사용자가 보드 접속
2. 전체 회의실의 예약 상황을 시간대별로 확인
3. 예약된 셀 호버 시 예약자/회의 목적 미리보기
4. 필요시 예약된 항목 클릭하여 상세 정보 확인

## 5. UI/UX 요소

### 5.1 메인 화면
- 헤더: 프로젝트명, 현재 날짜/시간
- 메인 콘텐츠: 예약 현황 보드 (그리드)
- 사이드바/필터: 날짜 선택, 회의실 필터 (추후)

### 5.2 색상 전략
- 빈 시간대: 밝은 녹색 또는 흰색
- 예약됨: 파란색 또는 주황색
- 현재 시간: 빨간색 강조
- 호버 상태: 밝은 강조 색상

### 5.3 반응형 디자인
- 모바일: 스크롤 가능한 테이블 또는 카드 레이아웃
- 태블릿/데스크톱: 전체 그리드 표시
